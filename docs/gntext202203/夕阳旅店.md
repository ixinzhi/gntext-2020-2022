# 夕阳旅店

作者：zxc20010506

TID：32705

# 1

夕阳旅馆一我是十年前去美国加州那家旅馆的。当时开车在公路上，漫天舒卷的云彩带着奇炫的色彩。天色将沉。空气中满是温溽的气息。我喝光了车上的水，口渴难耐。我记得下车前看了一眼远方的岩山。山那边有着炊烟。山与天边画出一道金边。天边的云似金，似紫。我琢磨不定。但我记得那一定是一种很奇特的颜色。我朝那家旅馆走去。带上了我的手枪。二“给我来一份牛肉炖菜”我还要了一杯冰威士忌。女酒侍叫黛娜。对，没错。是这个名字。肌肤雪白，金发碧眼。穿着连衣裙。“好的，警官”黛娜笑着说。我打量了一下左右。店侍就黛娜一个人。还有两位男士，一位女士。“先生，您见过这个人吗”“他是我们胡桃镇的一个小伙子。前些时日失踪了。”“哦，没有”“女士···”“不知道”“······”我叹了一口气。回到自己的座位上。艾熙是我们小镇上一家医生的儿子。从小受大家的欢迎。那小子小时候总希望跟着我跑别的镇。我记得我想起来这些事情是温暖的。可他最近不知为什么离家出走了。男孩子大了就该出去闯一闯嘛。可为什么不告诉父母一声呢？······“辛德警官，这是您的炖菜和威士忌”“哦，谢谢”我边吃饭边喝酒。不时看一下远方奇炫的夕阳。我瞟了一眼店里的钟。现在是“**六点**”“咚——————-”三“黛娜，你最近见过艾熙吗？”“三天前见过···他看起来好像不太好。像种了邪术一样。嘴巴里当时不知道在念叨些什么。可怕极了···”“他开车了吗？”“没”好。他走不远。但这也让我更担心了。这四周都是沙漠。他能走到哪去。“他往哪儿走了？”“山的那边”“······”黛娜走开了······四天色很晚了。我在旅店下榻。手枪被我枕在枕头底下。一点烛火。“呼！”吹熄了火焰，人当睡去。······房门被悄悄推开。“谁”我以把手枪上了膛。其实我隐隐猜到了。只有她有钥匙。我惊呆了。黛娜全身一丝不挂。雪白的裸体展现在我面前。“黛娜，你干什么？”“辛德警官，艾熙已经死了······”我已经几乎要扣下扳机。眼睛已经涨红了。“是谁杀死他的”“他自己去向了山的那一边”“什么意思”“他与我们木榴花姐妹会签订了自由契约。去往山的那边了。”木榴姐妹会是当地传闻存在的一个女巫会。看来艾熙这小子是被那群疯子害了。突然。一丝治安官的直觉突然闪过。这是有蹊跷。我松了一口气。放下了枪。“黛娜。你可以坦白告诉我。你跟艾熙是不是相爱了？准备私奔。”我觉得他们两个可能在演戏。因为女巫会只是传闻。数年来被举报也未查出什么来。来报案的人都像疯子一样说在迷幻中看到白塔。然后同伴一个一个被撞击成碎肉。后来移交案子。刑事警长说只不过是那个人发疯杀了他的所有家人。把尸体藏了起来。一个混蛋想逃避制裁瞎编出来的谎话罢了。当时虽觉证据上略有不足，因为毕竟没有找到尸体。但那个人是最大嫌疑人。去年已经是被处以绞刑。“辛德警官，不是的。但艾熙真的已经死了。”“我问他是不是真的想到山的那边去。他说是的。还跪下求我。脱下我的高跟鞋。把我的脚一边又一遍的亲舔。我当时还有些生气，但那些人还没接走他··我就随他去了。”黛娜向我展示了她的那双美丽雪白的脚。那脚掌在夜里的微光下好似维纳斯的一般···极美无比。“山的那边的人。都会跪着亲舔着双脚。辛德警官。我在这里是一个女侍。可我在那边就是神。州里的长官也好。对我来说，就像蚂蚁一样，只会在我脚下蠕动求生······”“只要我想，我就能轻松碾碎他们。”黛娜做出了一个轻踏的动作，然后用她那性感的脚趾碾动了一下。“噗嗤”“他们就会以这样的声音爆裂开来。”黛娜生出了媚美的微笑。我恍惚间扣动了扳机。“嘭！”五我忽然醒来，一身冷汗。原来是一个梦。幸好是一个梦。牛肉炖菜还热气腾腾。威士忌已然喝了一半。看来是太疲惫了，睡着了。“黛娜····钱放这了“黛娜还是之前的模样。我出门看前看了看那山外的夕阳。依然绚烂。我迎着干燥的风走向警车。我坐上车。打开广播。却发现播报的时间是”五点半“我急踩油门往小镇方向跑去。六山的那边金紫绚烂。云彩如飞。
