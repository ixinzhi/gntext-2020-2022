# 【总结】文章翻译《こびと専門SM嬢エリカ》


-   艾丽卡对小人们的行为和态度
    1.  小人们因艾丽卡巨大脸庞发出悲鸣，艾丽卡觉得可爱。
    2.  艾丽卡大声回应小人们的哀求，称赞他们了不起。
    3.  艾丽卡用大音量伤害小人们，并称其为奖励。
-   艾丽卡对小人们的残忍行为
    1.  艾丽卡用冲击波击碎小人们的身体，称其为好吃。
    2.  艾丽卡用尿液冲刷小人们，让他们筋疲力尽沉入尿湖。
    3.  艾丽卡对已无生存者的房间自言自语，表示痛快。
-   艾丽卡与老板的冲突
    1.  艾丽卡因客死而向老板诉苦，请求留下。
    2.  老板指出艾丽卡的行为导致客死，收益无法提高。
    3.  艾丽卡无言以对，被老板要求离开。
