# 1

人物来自于「天才王子的赤字国家重生术」

　　人物介绍：妮妮姆

　　拥有通透亮丽的银白长发和有如火焰般的红色眼瞳的少女，弱小国家纳特拉王国的王太子维恩的首席辅佐官[1]、从小就侍奉他的贴身随从。

　　故事发生背景：安托加达尔侯爵来访，即动画第四集

　　「你小子还是很懂道理的嘛，快让我见见我的洛薇蜜娜宝贝」格拉多说道

　　「你小子?」刹那间，妮妮姆的眼神充满了杀气，双手握紧，仿佛做好了随时杀死格拉多的准备，因为...他侮辱了维恩。

　　维恩见状，赶忙阻止了妮妮姆，以免她酿成大错。

　　「格拉多殿下，我已经为您安排好了住房，为何不先享用一餐，做好打扮再去见洛薇蜜娜皇女呢?」维恩说道

　　「很有道理，那就请带路吧，小子」

　　「?!」

　　格拉多不知道，自己今晚的种种举动，将会给自己甚至国家招来杀生灭国之祸。

　　在享用完晚餐后 格拉多与洛薇蜜娜皇女约定好了明日将会前往帝国提亲，届时也请维恩殿下做见证人。

　　在格拉多走进自己房间的那一刻，他被震惊了。

　　映入眼帘的是自己的国家“安托加达尔”

　　「我什么时候回来的，明明我刚刚还在纳特拉的啊」

　　环顾四周，街上空无一人，再怎么傻，格拉多也感受到了诡异，决定立马回头，走出房门。

　　然而，房门已经不见了。

　　格拉多顿时害怕了起来，他开始四处走动，想找户人家问问情况。

　　「哼哼~真是一只丑陋的虫子呢~」说这话的正是妮妮姆，此时的她正坐在维恩的腿上，饶有兴致的看着地上被缩小的安托加达尔的模型。

　　「妮妮姆...我说....这样真的好吗，万一他们发现了，岂不是会举兵进攻纳特尔，到时候我可怎么办啊。。」

　　「全部缩小踩碎不就好了，这种背叛的国家，也没有什么存在的必要...还是..维恩殿下，你想替格拉多辩护一下吗?」

　　「不....不...我没话说，确实，有你在，没什么好担心的。」

　　拥有全大陆最强的魔法（我还没看原作小说，所以不知道这里有没有魔法，如果没有的话，就当是架空好了 谢谢谅解）

　　缩小/巨大化的魔法，虽然是弗拉姆人，但妮妮姆完全拥有改变整个世界的能力，只不过。。。是毁灭。。

　　所幸她兴趣并不大，她只会对侮辱自己亲人，和侮辱维恩的人或国家出手。

　　而格拉多，很不幸，将要成为第一个死去的“虫子”了

　   维恩猜到了接下来会发生什么，于是便跑回了自己的卧室。

　　妮妮姆锁上了门。

　　「那么，游戏开始了」

　　格拉多还在不断地寻找人影，然而，还是无果。

　　「这到底是怎么一回事啊」

　　「结界解除」

　    突然，一阵巨响传来，吓得格拉多赶紧趴在地上，等没什么动静了，才爬起来。

　　「怎么回事...刚刚那声响声」

　　「看看上面噢~小虫子」

　　「这声音是!果不其然，是妮妮姆。」

　　只是...此时的妮妮姆，相当于格拉多的10000倍，仅仅是妮妮姆的皮靴，就已经成为了遥不可及的存在。

　　「啊啊啊啊啊，怪物，是怪物!」

　　「阿拉，说怪物可真是侮辱人家了呢，你不应该想想自己的处境吗」

　　说完，妮妮姆便踏出了她第一脚。

　　轰!

　　纵使妮妮姆只是轻踩，建筑物也无法抗住这巨大少女的任何一个动作。

　　「安托加达尔的建筑也不过如此吗，连一个16岁的女孩子都抵挡不住，真弱呢~你说对吧，格拉多小虫子」

　　然而此时的格拉多已经完全听不进妮妮姆的话了，恐惧遍布全身，他开始一味的奔跑，祈祷着自己能够逃离这个地狱。

　　然而这在妮妮姆看来就如笑话一般，格拉多也许永远不会知道，就算自己跑出了城市，也还是在纳特拉王国的...地板上。

　　「跟维恩比起来，一点也不可爱呢，一味的乱跑，维恩还会用用脑子，虽然也没有什么用是啦~那么，游戏正式开始啦」

　　又是一步，妮妮姆的每一步，都会有大量的建筑物被埋在鞋底，宛如沙子般脆弱。

　　「真是垃圾，这样下去的话，游戏很快就会结束呢，缩小一点好了」

　　光芒四射，妮妮姆此时变成了100倍的大小。

　　「这个大小就很方便玩了呢。那么，虫子在哪呢。」本以为格拉多会找个地方躲起来，妮妮姆都已经做好了精神探视的准备。然而....格拉多在大道上...奔跑着。

　　「真是笨啊，这种人是怎么成为领袖啊。」

　　不费吹灰之力，即使只是100倍，妮妮姆也很轻松地追上了格拉多。

　　「啊啊啊啊!」

　　砰!

　　妮妮姆将皮靴踏在了格拉多的面前，格拉多来不及停下，径直撞了上去。

　　「啊拉~这么喜欢我的鞋子吗?要不要舔舔看呢~」

　　「你...你个恶魔!我警告你，你最好现在立刻马上让我回去，否...否则...纳特拉..将会遭受灭顶之灾!」

　　「欧~真有趣呢，灭顶之灾，不如你先想想你自己的处境?」

　　妮妮姆抬起脚，对着格拉多踩去

　　「啊啊啊!不要...不要!」

　　「诶?」

　　皮靴没有踏下，而是恰好的停在了空中，停在了格拉多正好能看见刚刚的建筑残骸...

　　「小虫子，我再问你一次，舔还是不舔」此时妮妮姆的语气已经充满了威压

　　「我....我舔 求求你 放过我吧」

　　「真乖，小虫子，只要你把我的鞋舔干净，我就放你走。」

　　「真...真的吗!」

　　「当然是真的，不过，不是这个大小噢」

　　妮妮姆再次变回了10000倍的大小

　　此时的靴底，对于格拉多来说，已经是一片天空了。

　　「这....可恶，等我回去了，定让你们生不如死!」格拉多内心想到

　　然而，正当格拉多走到了靴子前，准备开始舔的时候，靴底突然远离了自己。

　　然后...

　　砰!

　　大片大片的街区被毁灭，并且随着刚刚踩下产生的震动，损失还在不断扩张

　　「真以为自己够格舔鞋吗，哼~」

　　「真无趣呢，一点感觉都没有。」

　　说完，妮妮姆变回了正常大小（原本的大小，模型是被缩小的）

　　「再见了，虫子~」

　　轻轻踏下，整个国家随即消失。

　　回到卧室

　　「这么快就结束了?」维恩一脸疑惑地看着妮妮姆

　　「没什么意思，没有脑子，漫无目的，完全没有兴趣呢。还是维恩你比较好玩一点。」

　　「啊...妮妮姆，你不会又要...我还没从上次走出来呢...」

　　「安啦，等把洛娃的事情解决先吧，不过，只玩一个人没什么意思呢，下次，就玩一个国家把~」

　　用最温柔的语气，说着最恐怖的话的妮妮姆。

　　会给整个大陆带来什么呢?
