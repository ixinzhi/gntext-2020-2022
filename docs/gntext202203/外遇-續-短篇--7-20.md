欣羅的身體比母親大了幾圈，就連胯部肌肉收縮後對奈米城市造成的損傷都是數倍成長，螢幕不斷傳來各地死傷的畫面，加上那個貌似蒙汗的男子驚恐的樣子，欣羅的情緒被調動了起來。

“不弄死你就好”

蒙汗只來得及聽到這句話，整個城市就劇烈搖晃，大樓紛紛瓦解，車輛翻覆，行人被壓的支離破碎。

“以前都沒有專門用奈米級的城市來玩，至少都是公尺級或公分級，沒想到只是坐在這邊，他們就要死光光了…”欣羅看著畫面中地獄般的場面居然越來越興奮了，尤其是蒙汗抱頭鼠竄的樣子讓她有點罪惡感，但更多是刺激，而這讓她的反應越來越大，腦中突然回想起當初在奈良國跟蒙汗一起看電視譴責新聞播報的殺人犯，那此刻自己算什麼？

這已經是屠殺了吧！

可是什麼都沒做啊！

就是安安靜靜的坐著，下體的生理反應而已，就快將這合計超過數億人口的城市群給顛覆。

欣羅直到此刻依然忍著不伸手指進去，因為一旦真的伸進去，底下幾億人絕對不夠死的，這真是考驗意志力，還有對底下老公的愛意。

….

回到奈良國後，欣羅發現蒙汗態度大幅度改善，起初還覺得可能只是幾天的熱度，但直到一年後都沒有再被酸言酸語，那態度叫做一個好。

對於那天欣羅胯下城市群的災難，即使心有猜測，兩人仍心照不宣，過著甜蜜和諧的生活。

全篇完結。
