# 女神诞生记

作者：wuliaojianke

TID：28935

# 1

一个阳光明媚的青春，一个写字楼。电梯打开了，是她，画着淡妆，涂着粉色的唇膏，清淡却不失清香的香水味，冲我微微一笑。  将近一米八的身高、曼妙的身材，年龄不算很年轻，却保养的那么好。

嗨~对于美女的微笑，我也友好的打着招呼。谁会拒绝这么一个漂亮的美女呢。

我刚想说出她的名字，但是，此时这个美女，应该是女神，却在我的身旁，看着敲击键盘的我，提醒着：亲爱的，女神的名字是不可以直接说出的哦。

是啊，很快，整个世界，就要进入女神的统治了吧，一个新纪元。

是啊，谁能想到，每日和我出入同一个写字楼的这个美女，竟然是至尊女神呢。她的体型巨大到人类无法想象，拥有强大的法术。

那么，就从女神的觉醒开始讲起吧。

# 2

那是一起出差的日子。忙完公务，我收到了她的讯息：要一起出去走走吗,喝一杯。我答应了。

我在酒店门口等她。此时的她，穿着休闲装， 露出爆乳，是那么美丽。我的那里不由得有些反应，但还是忍住了内心的想法。但这一切，怎么逃得过女神的眼睛呢？

# 3

伴随着夏日的凉爽，我和丽走在河边上。此时的她，穿着性感，婀娜身材，让人浮想联翩。但我告诫自己，绝对不可越界，因为我们就是同事关系，况且，她还是个有夫之妇。但是，没有人能够抵挡女神的诱惑。丽的体香和香水，吹拂在我的身体上。我那里很自然的石更了，这是一种自然的生理反应，自然被她看在眼里。

她牵住我的手，我没有抵抗，就这样，我们自然的走在了一起。过了几分钟，她突然回头，朝我摸摸一下：帅哥，感觉如何啊。

我的脸顿时红了，有些支支吾吾的说到：丽…你，真是，真是太美了。这时的丽微微一笑，抬起双手抚摸我的脸颊：那，你喜欢我吗？我有些犹豫：喜欢，真是太喜欢。 不过，你已经，已经…

随后，丽说出了那句让我永远记住的话：帅哥，你喜欢女巨人吗？

# 4

我有些惊讶：你，你说什么？什么女巨人啊，什么意思。

丽继续妩媚的看着我，抚摸着我的脸颊：呵呵，不要再掩饰什么了。需要我解释吗？女巨人，GTS。或者，那个你经常上的论坛？需要我进一步解释吗

我有些惊讶：你，你怎么知道的，难道，你也是同好吗?

丽冷笑了一声，摸到了我的胸部：同好？随你怎么理解吧。那么，你就不认为这个世界上真的有女神存在吗？说着，她吻了我的额头：如果那个女神，是我呢？就这样，我们紧紧吻在一起。

# 5

我们热吻了几分钟，然后打上汽车，回到了住的酒店，脱去了彼此的衣服，开始了缠绵。就这样，我们享受着激情，缠绵在一起。我尽情的享受着和女神的性爱，不知道抽查了多少次，然后不知不觉进入了梦乡。

过了不知多久，我睁开了双眼，这、这是哪？  

# 6

怀着一丝睡意，我揉了揉双眼醒了过来。这里不是我们所住的酒店吗？我回想着刚刚发生的一切，在外边，我们回到了这里。开始缠绵。

嗯，已经是清晨了。我望着窗外，那么，丽去哪了呢。正在这时，大地发生了巨大的震动。轰隆，轰隆。地震？不，地震的声音不是这个样子的。不知道从哪里来的感觉，我朝窗外望去，一个美丽的胴体若隐若现，像...那，那是？整个酒店和城市里陷入混乱，大家纷纷疏散，我却来到消防通道，朝着楼顶跑去。来到天窗，那个胴体越来越近，前方的房屋也被踩碎了。我才看清，那胴体，穿着美丽的低胸晚礼服，爆乳若隐若现，分明是一个女子的模样，那个女子正是...丽？

轰隆轰隆，脚步越来越近。此时的丽，估计有一百多米高吧。天啊，难道，GTS是真的存在吗，我扑通下跪。整个世界秩序，要被改写了。
