# 1

只是試寫一下

大概會寫個7篇

1.輕度罰則  (洗腳/打掃教室)(50%)

2.中度罰則

「阿真是雖，早知道不要聽同學的話了，去做什麼蠢事了」

男主走到了學生更衣間外  

此時面前出現了二個魁梧的男學生 好高... 估計有200公分吧

(os:原來是要幫這兩個巨人洗腳 光是想到就害怕 )

「我是來做 那個…… 懲罰的」此時男主嚇的有點說不出話

「你太早來了喔 在外面等一下 時間到會有人來通知你」

此時裡面傳來一個女生的聲音

「請進」

原來是女生來服務他們啊，還真是幸福，男主心想

偷看一下好了 ，眼前看到兩個同學拿了兩個超大的水盆。

此時出現的景象讓他嚇一跳

出現了比那些巨人還高的人

相比之下他們跟矮人沒兩樣

「喂矮子 在不快點我就連你的份一起享受了喔 」

又出現了另外一個女學生，確實比另一位矮了不少，但仍在兩位男學生面前還是顯的高大不少。

「唉呦別說了 你什麼都不用做都可以長到快3.5米 真令人羨慕 」

隨後她們坐椅子上並把腳放進大水盆等男學生來服務

「呵呵 這次要洗乾淨一點喔 」

兩位男同學開始洗女學生的雙腳

女學生巨足是男同學的手兩倍大

讓洗的過程十分困難 。不過在這之前她們就已經洗好澡了，只是配合學校的規定罷了 。若女學生不想做的話也可以直接放他們一馬並分配給他們一些更輕鬆的任務。

但某些人會趁這個機會好好折磨一番……

「怎麼了 連我的腳都抬不起來嗎，

男學生使勁吃奶的力氣，但女生只需要一點力量就能將其抵消。

甚至把手壓倒水盆底，令男生十分難受。」

「受不了，就只好勉為其難的幫你一下吧。」女生不再繼續施力，但腳的重量還是非常的重，讓男學生連連喊累。

而另一位較矮的女生用一隻腳跟男生打起單方面的水仗，而男生除了噴了一身濕之外，還要閃躲迎面而來的攻擊，被女同學的大腳踢中可不是鬧著玩的。

「看招! 哈哈哈哈 」

男同學在慌亂之中跌進水盆 在盆內掙扎

「救援到嘍~」女學生彎腰用手撈起來。

難以想像2米高的男生有這樣遭遇，不過對方可是有超過3米的身高的。

「喂 還有另外一隻腳 」較高的女同學說道。

隨後就用腳抹在男同學臉上

弄的一身濕並跌在了地上，後起來替女學生的腳洗乾淨

男學生們出來的時候已經累倒在地無法動彈

「喂這邊 此時隔壁的門出來一個人 叫住了他」

他看了看才發現自己走錯了地方

但心中恐懼已經佔據了整個身體

走都走不動。

深怕因為偷看了女子更衣間而被判更嚴重的懲罰

進去隔壁房間後看到了比剛才略小的擺設 ，

並一眼就認出眼前的女孩是誰。

「你是風紀委員 。」

風紀委員露出輕輕的微笑

「呵呵 你可要好好的服務我喔。」

眼前的這名女孩就靜靜的坐著讓他洗腳 她認為才輕度懲罰不需要太嚴格 兩人聊了關於學校的事 時間飛快的過去，洗完把水倒掉就回家去了。

輕度罰則確實是不需要太過火

學校甚至有規定下手太重反而會讓自己受罰。

註:風紀委員專門在一年級以及二年級校區活動 專門負責管理學生一些行為 身高通常在2.3到2.75公尺之間

以便在校園行動自如又能起到管理的效果 若在任內身高長到太高的話會被調去巡三年級校區。」

註:「此世界的女性到15歲之前還會長高」
