# 至尊女修

背景设定：这里是修真世界，体内灵力越多体型增幅也就越大。而个人实力主要看三方面，一是体型二是修为三是斗法。受到世界规则影响，世界中女性修士普遍丹田容量更大，因此同阶下体型增幅也会大于男性修士，坏处则是需要更久的时间来积累灵力，同样的天赋和资源下常常是男性修士先晋级。不过一般来说晋级的最大难关在于境界屏障而不在于灵力积累，因此同样天赋下女修的实力总是可以后来居上。长期实践总结出规律后，在同样的天赋下，女修往往会被分配到更好的修真资源。

云梦出生在青云界云城云府。这个家庭由父母、一个男孩和三个女孩组成。老大是男孩，其余都是女孩，云梦正是老幺。

云梦的母亲名为云青，是一名武道先天高手，也是云府的府主。她是云府中修为最高的人，同时也是身高最高的人。十个月前，云青忽然感觉自己周围的灵气变多了，虽然不知道是什么原因，但总归是一件好事，她开始利用这个机会尝试突破困扰她很久的宗师境界屏障。

过了一周，云青表感知到自己怀孕了，武道先天的高手感知比起普通人要强的多。联想到自己身边一周前开始出现的灵气聚集现象，隐隐猜到这个孩子的不凡。

过了几个月，云青验证了自己的猜想。自己怀前几个孩子的时候修为身高都不如现在，但体内的胎儿的占位感明显高于前几个孩子，这说明她的体型要远远超出常规胎儿。人的基础身高是有上限的，超出这么多，说明她受到了灵力增幅，胎儿是不可能练武的，很可能是已经步入了相当于武道先天的练气期。云青对于身材高大的先天高手来说，怀孕对生活并没有什么影响，但这次怀上云梦却让她感受到了些许无修为孕妇的感觉。

既然胎儿时期就能自己修炼，那么说不定也有了自己的意识。抱着这样的想法，云青开始和肚子里的云梦交流。说是交流，其实是云青单方面说话，不过聪明的云梦也会在子宫内活动以做回应。

这天，大儿子云凡和二女儿云凤练武回来了。自从云青怀孕后便不再亲自教他们练武，他们也对家族的新成员很好奇。大儿子云凡十六岁，人如其名天赋不高，跟着母亲练武多年还是待在锻骨期。二女儿云凤十二岁，练武天赋比云青还要高，又非常要强，锻炼刻苦，经常会得到云青的夸奖。除此之外还有个十四岁的大女儿云雪，但她检测出灵根后就去了修真门派，不在府里。  

# 2

云凡偷偷瞄着妹妹云凤的头顶，打量着她与自己之间的身高差。随着云凤练武的进步。他俩之间的身高差距是越来越小了，也许再过一段时间自己就会被超过。 云凡心中苦涩不已，他本身并不算特别矮，初始身高一米七的他加上锻骨期的灵力加成，身高足有两米。但云凤身为女性的灵力容量更大，身高加成也更多，要不是年纪还小，初始身高不够，恐怕已经高出自己一头去了。 另一边云凤的心情就舒畅多了，她才十二岁就达到了锻骨期巅峰，等突破了炼脏期，她就能远远把哥哥甩在身后，成为年轻一代的最强者。云凤的偶像是母亲云青，梦想是超过母亲，她每天都会为这个理想坚持刻苦练武。 云凡云凤二人来到云青房间内，走到母亲身前，高出侍从一头的他们在云青面只能堪堪达到膝盖位置。云青将二人抱至自己面前，与之交流。 云凤凑到云青耳边，神秘兮兮的对着云青说道：“妈妈，再过一段时间，我要给你一个惊喜。” 云青看着女儿，笑意盈盈，身位武道先天巅峰的她自然能看出来女儿快要突破炼脏期了。云青眨了眨眼睛，笑着说道：“妈妈也会给你一个惊喜。”经过这段时间的修炼，她感觉到困扰她多年的瓶颈已经松动，她有预感，云梦出生之时，就是她突破宗师境的时候。 云青叮嘱二人要勤加修炼，这段时间她要专心闭关，出来后会检查练武进度，如果让她不满意，会给予二人惩罚。 云凤好奇的看着云青的肚子，那里孕育着自己的妹妹。云凤摸着云青的肚子说道：“等到小妹妹出生了，我要教她练武。”云青微笑不言，年龄太小的孩童并不适合练武，而且这个“小妹妹”可不小。 待到云凡云凤离去后，云青找来自己的副手云岚交代了些事情，便进密室闭关专心修炼养胎了。 十月怀胎后，云梦顺利出生在了密室里，汇聚而来的灵气助力云青直接冲破宗师境的屏障。顺利突破的云青双手捧起眼前的小人。喜爱至极。说是小人只是对云青来说的，对于普通人来说可一点都不小，刚出生的云梦就有一米二高，将汇聚而来的灵气尽数吸收后突破到了练气二层，身高达到了一米七。云梦虽然只是刚出生，但已经脱离了幼儿状态，一副女童模样。云青对这个女儿有了新的认知，云梦吸引来的灵气能直接助力她突破宗师境，但云梦吸收掉大部分灵气后只突破到了练气二层，要知道小境界和大境界所需的灵气可是相差甚远。云青心里盘算着，云梦的天赋相当之高，但她的丹田容量过于恐怖了，修炼的也未必有多快。 云青将一个女孩领到云梦面前，说道：“从今以后，这个女孩就是你的奴隶了，她会担任你的贴身女仆。” “妈妈，奴隶是什么？” “奴隶是专属于你的‘东西’。你是强者，她是弱者，所以你可以对她做任何事，包括成为你的奴隶或者是宠物。” “宠物又是什么？” “宠物嘛，”云青组织了一下语言，“就是你的东西里最喜欢的那个，所以相比其他的东西，你可以给她一些特权。” 云青所说的正是这个世界中大多数人所认可的世界观，实力就是一切。 “我先走了，你去玩吧。”云青要去稳固一下自己刚突破的境界
