    2.  上位存在受世界法则限制，不能在人群密集处战斗。
    3.  某建筑物地下室成为奢华浴室，展示上位存在的生活。

    2.  蕾琪魔力被吸收，身体被奥希尔封印在体内。
    3.  奥希尔能将蕾琪的脸和乳房长在自己身上，作为象征。
-   蕾琪的羞耻与挣扎

    2.  奥希尔提出条件，要求蕾琪发出喘息声以换取小便。
    3.  蕾琪心理防线动摇，身体被奥希尔控制。
-   奥希尔对蕾琪的调教与享受

    2.  奥希尔通过挑逗蕾琪获得更多欢乐。
    3.  奥希尔提出条件，要求蕾琪发出喘息声。

    1.  蕾琪头部被奥希尔下体融入，成为排尿口。

    3.  奥希尔享受穿着丝袜和鞋走路的感受。
