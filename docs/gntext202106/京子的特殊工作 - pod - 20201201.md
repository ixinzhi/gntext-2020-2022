# 1

“哎京子京子，为什么你这么早就回去了呀，明明还有那么多时间的。”

　　名叫京子的少女摸了摸头，苦恼道：

　　“唔，我最近在打工。”

　　“哎，京子好厉害，明明才是国中生呢就开始自力更生了。”

　　“唉嘿嘿，也没那么厉害啦，是很轻松的工作哦，亚美要不要也来试试呢。”

　　“唔……”

　　亚美挠了挠头，似乎有些犹豫：

　　“还是算了吧，最近妈妈替我找了课下辅导，时间根本不过呢……”

　　“啊，好可惜，我的工作实际上还很有意思呢。”

　　京子收拾好了书本：

　　“那么明天见亚美。”

　　“明天见京子。”

　　……

　　“我回来了。”

　　亚美打开房门，不出意料哥哥又没有在家。

　　“总是这样，哥哥好过分，又留京子一个人在家。”

　　虽然知道哥哥是在为了养活她而努力工作，但是还是忍不住责怪他，就不能陪陪自己吗。

　　自己现在也有工作了，要不要告诉哥哥？

　　算了，哥哥应该会担心的吧，毕竟她的工作，很奇怪。

　　京子脱下鞋子，就在这时她口袋里的手机震动了起来，京子拿出来那块屏幕很大的只能手机。

　　“唔，新任务来了呀，哇，这么多赏金。”

　　“三万日元呢。”

　　京子这种年纪的人是很少拥有智能手机的，更何况京子的手机还是很高级的。

　　这是委托方给她的手机，就是拿来发放任务和赏金的。

　　京子快步跑到了自己的房间，不出所料，她的房间里面全是像是模型一样的建筑物。

　　地板上，桌子上，她的床上面，甚至拖鞋里面，还有她昨天忘记拿出去的穿了一天的袜子里面都充满了这种像是模型一样的小城市。

　　与其说是小城市，不如说是将整个城市缩小后整整齐齐的摆放在她的房间里面，似乎是因为京子没有打开灯的原因，那些城市灯火通明，从京子的目光中看过去就像是一个个小小的，在散发着光芒的萤火虫一样。

　　“嘿嘿。”

　　京子一只脚慢慢伸进去，由于整个房间都布满了这种小小的城市，她那双穿着白色布袜运动了一天的有些小小臭味的小脚还是踩到了楼房中。

　　“啊，抱歉抱歉。”

　　京子毫无歉意的道歉，虽然不止一次了，但是每次脚底接触到城市那种轻微的触感，那种平日里只有抬头才能看到了建筑物被自己一个国中生一只小小的脚就能踩坏的感觉真的是太让她着迷了。

　　那种脆脆的感觉就像是在给她按摩一样。

　　那些居民楼还没有自己的脚趾头高呢嘻嘻，小小的看起来真可爱，好想，好想……

　　将它们都踩碎呀……

　　城市里面的小人似乎被刚才那“巨大”的声响惊扰到了，纷纷骚动起来，只不过这些骚动的声响传动京子耳朵里面更像是嗡嗡嗡的蚊子似的。

　　里面有不少探照灯纷纷将光指向天空，照在京子身上。

　　京子好奇的看着自己身上的那些小点点，这些可爱的小东西总是能给自己带来不同的体验呢。不知道在他们眼里自己现在是什么样子呢？估计是一块大大的墙吧。

　　这时候京子的手机忽然震动起来，京子打开手机，上面显示着京子此次的工作。

　　“唔，又是处理人口过剩的异界城市吗，咦？今天是我的生日吗？哇还要给我生日礼物。”

　　“可是礼物在哪里呢？”

　　京子向周围好奇的看了看，出了灯火通明的小小城市就没有其他的了。

　　“算了，先完成工作吧。”

　　京子打开了墙上面的灯开关……

　……

　　躁动，不安。

　明明高高还挂在天上的月亮下一秒就消失不见了，天上面一颗星星也没有，没有风，也静悄悄的，整个天地都仿佛瞬间安静了下来，反倒是整个城市充满了一股奇异的香味。

　　还有天上，被探照灯照射出来的一面高大的墙……

　　忽然，似乎瞬间就来到了白天，整个城市亮堂起来，人们纷纷抬头看着天上的太阳，这个太阳，看起来怎么这么像日光灯？

　　这时候城市里的人们似乎发现了他们身边的异样，那是，玩具熊吗？怎么这么大，那是凳子？为什么也这么大，

　　还有天上这仿佛望不到边的两根肉色柱子。

　　这好像是……

　　一个小女孩的腿。

　　这是什么？虚拟投影吗？那个公司的新产品选场吗？

　　人们有些不安，开始相互推挤，短短十几秒钟就已经发什么成百上千起车祸。

　　“唔，咳咳，小虫子们晚上好呀。”

　　天上出来一阵天真的声音，虽然是听着很童真的小女孩的声音但这然不少人痛苦的捂住耳朵，他们的而耳膜快被震裂了。

　　天上，出现了一张巨大的脸，那是一张稚嫩的国中女生的脸，巨大的少女笑盈盈的看着他们。

　　那眼神不像是在看人，更像是小孩子看蚂蚁……

　　“咳咳。”

　　少女仿佛想到了什么：

　　“小虫子们，从现在开始我就是你们的主人！你们都要听我的。”

　　天上的少女脸微微发红，似乎有些不好意思。

　　京子还是第一次和这些小虫子交流，以前为了钱她可是很快就摧毁了城市的。

　　“开什么玩笑！”

　　“究竟是那个公司的无聊产品，在这样下去我就去投诉了！”

　　“滚开你这个臭**”

　　居然被拒绝了？还被骂了？

　　京子不敢置信的看着自己脚底下那些人，他们居然敢那石头扔自己，虽然京子没什么感受，但是……

　　果然和这群虫子没什么好交流的吗。

　　“京子生气了哦。”

　　京子抬起自己还穿着布袜的脚，慢慢地向着人群最多的商业街落下去，虽然他也很想用力踩下去，不过这里可是自己的家呀。

　　白色的布袜和商业大楼接触，钢筋混凝土铸就的大楼根本无法抵御柔弱的布袜，繁荣的商业街因为京子的一脚瞬间变成一团废墟，在大楼里的人们根本来不及跑出去就被京子的小脚碾成肉泥，和破碎的大楼永远链接在一起，被踩碎的混凝土围在京子的小脚周围形成了一个巨大的脚印，当然，对于小人而言。

　　京子不自觉露出一个微笑，果然这种蹂躏人类千百年成就的感觉无论多少次都不会感觉腻呢。

　　京子脚自己的脚抬到空中，被京子布袜上的线条缠住的水泥碎块跟个被带上天空，然后因为京子的抖动有重新落下来，布袜上还有一些人类死死抱着布袜的纤维，只不过太小了京子一点感觉也没有。

　　而被京子踩踏过的地方露出了她房间的地板。

　　“哎看到了吗？你们引以为豪的建筑还没有我家得地板结实呀。”

　　“这么脆弱的你们有什么资格拒绝你们主人的命令呢。”

　　京子故意将自己的脚将抬到空中划过居民区的楼房，然后放在了居民区，果不其然，被京子脚掌下面传出了一阵微弱的尖叫。

　　京子慢慢放下了自己的脚，小巧的脚掌轻而易举地将脚下面的一切摧毁，脚掌落下前期一股风浪携着破碎的混凝土碎块向远处飞去，将周围以为逃过一劫的人砸成碎块。

　　“唔，有点痒呢？”

　　京子忍不住动了动自己的脚趾，被布袜包裹着的脚趾像是巨大的猛兽一样将脚趾前面还没有被踩到的居民楼连着里面的人一同吞噬。

　　京子的脚趾轻轻夹了夹，被脚趾包裹的楼房瞬间被碾成粉末，慢慢顺着京子的布袜缝隙滑落，落到地上形成一个沙丘，一个比这些楼房还要高的沙丘。

　　“哎，你们想要逃吗？不可以哦，这里可是京子的家呀。”

　　京子看见公路上堵着的车。

　　京子又一次抬起脚朝着早已堵成长龙的道路踩下去，地面上的汽车喇叭狂按，甚至有不少人驾驶者车开始乱撞，但是这都逃不过京子那小小的脚掌。

　　人们绝望这看着天上那原本洁白现在却灰黑一片的脚掌，布袜上面挂满了破碎的城市碎片，还有猩红的小点点点缀在上面。

　　“嘻嘻，怎么不跑了。”

　　京子将自己的脚掌放在了这些车的上空，一些原本就被挂在上面的建筑碎块不受控制落了下去将这些堵死的车辆砸成铁饼。

　　“哎呀呀，举了这么久有些累了呢？”

　　脚下瞬间传来了细微的叫声。

　　“咦？你们在说什么吗？哎呀呀听不到呢。”

　　下面的声音更加大了，不过对京子来说还是和蚊子似的没有什么区别。

　　京子将头放低侧着耳朵认真听着，她喜欢听这些小人们一起发出的尖叫，这酥酥麻麻的声音让他的耳朵十分舒服。

　　“嗯嗯，哥哥说努力的孩子是会得到回报的哦。”

　　京子没有将自己的脚落下去，她脱下了自己的布袜，然后随意的扔到了这条堵得长长的车龙上面。

　　“奖励给你们的，是国中生刚刚脱下来的袜子哦，我今天可是认真跑步了的，变态小人先生一定会喜欢的对吧。”

　　“嘻嘻。”

　　刚刚以为逃过一劫的人类瞬间被一张铺天盖地的袜子遮住，上面传来一阵阵少女的汗臭味混合着奇怪的奶香，形成了一股怪异的臭味。

　　“咳咳，唔咳咳！”

　　袜子下的人们没多久纷纷咳嗽起来，一些体质差的更是昏倒过去，没过多久袜子上的汗水开始被汽车散发的热量蒸发，这些车辆里面的人瞬间都被少女的汗水蒸汽所包围，他们像是被汗水蒸笼在蒸煮。<title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
