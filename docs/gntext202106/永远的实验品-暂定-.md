# 1

第一章

“怎么突然想到带我去你的大学里面的了？”

我有些疑惑，毕竟她基本没怎么提过她搞科研的事。

“实验有成果了，我觉得这种好事应该跟你分享一下。”

钟兰眯着眼，牵着我的手拉了拉。

“说不定你也会感兴趣呢…”

两年前我和钟兰在大学里认识，聊得来就渐渐熟络，我是医学系生，她是搞科研的。虽然看起来我们像是情侣，但也只是在朋友之上恋人未满的程度，似乎更像是贴心的朋友。日常生活中我也会支持一下她的工作，她也很喜欢分享自己的事，只不过对自己科研的部分很少提起。

今天突然把我带到他的学部，说是给我参观一下他的研究成果，虽然有些疑惑，但更多的是好奇，毕竟和她相处这么久，我还从来不知道他到底是做什么的。

一大早就来到我家打车去了学校，看来是很要紧的事情呢…对她老说很重要吧…

“忙活了这么久，总算有了些成果吧…”

虽然钟兰嘴上这么说，但她的眼神一眼就能看出来。

他基本上是跑着的，拉着我快速的穿过其它校区来到了科研楼，学校把科研楼放在最里面的校区，使得这里似乎别别的地方更加幽静，应该也是人少的缘故，显得干净很多。

“昨天刚刚完成了这项实验，虽然之前进展就已经很明显，不过这一次应该算得上里程碑似的吧？”

钟兰自言自语的说着。

“智慧型的开发终于实现了呢…”

她用身上挂着的卡片打开了科研楼下的大门。随后我们沿着楼梯向上来到了第三层楼。接着她加快了速度。

“你们这一层楼都是实验室吗？”

我看着一旁的房间，里面摆满的器材。

“算是吧，这栋研究楼基本没什么人，也就我们几个人在搞这些东西了。”

我好像想到了什么，但还是没说出口。

随后她站在一个实验室的门口停了下来。

“到了，这就是我的实验室。”

她的脸上满是兴奋。

随后她打开了大门，一股冷空气扑面而来，我不住的打了个哆嗦。

“这里是…你的研究室吗…”

“是的，很棒是吗？”

她松开了我的手跑到了一个桌子旁边，上面摆满凌乱的仪器。

我也随意的走动，想看看她工作的地方。

“啊…那个东西请不要碰。”

我快速的收回我的手，看着眼前的瓶子，里面装着粘稠的液体。

“抱歉…”

“没事没事，你第一次来…嘛，看看我的作品，和你刚刚碰到的那个瓶子有联系哦。”

她提着一个箱子走进我，似乎里面装的什么东西，箱子在微微颤抖。

“这就是我这几年的实验成果，很厉害吧？”

她一脸兴奋的看着我，打开了她手中的箱子。

我的瞳孔收缩，心似乎停止跳动，她手中箱子里的，不是个人吗？

“啊…抱歉，我忘记解释了。”

她似乎看出了我的心思，轻轻地盖上了盖子。

“这是我们最新的实验成果，克隆和复制人体，且将其缩小化，主要用于医疗方面，可以将现代医疗无法解决的问题跳过，将意志存储后放入克隆人体中，实现医疗问题的跳跃…”

“当然，这些小小的家伙不是人哦，是属于无意识克隆人体，不存在意志，而且也不是人哦…”

她笑起来，把手中的箱子放在一旁的桌子上。

“我想着你可能会对这种东西感兴，趣所以特意把你叫过来看看。”

我确实很吃惊，这种东西确实可以给社会带来很大帮助，但似乎也是可以起到反作用的。

我夸赞了一下她，钟兰很喜欢被人赞赏。

但我却对缩小人体这个感到吃惊，到底是怎么做到的？

当然我也问出了口。

“这个嘛，记不记得你刚刚碰到的那杯药瓶？”

她指了指我身后的桌子，上面放着的药瓶闪着异样的光。

“具体内容我也不跟你多做解释，反正你也听不懂哈哈…”

她把箱子放回原来的地方。

“大概能缩减人体多余的物质将人缩小，这也是为了便于保存实验物品和储存更多的实验材料。”

她想了想。“很多时间我都想告诉你我做的这些，但是会长总是叫我们保密，不过现在实验有成果了，我相信会长也很愿意公布吧…”

她笑起来道，“当初我接到这个任务的时候也很吃惊，不过在老早之前我就有对这方面的发展，所以很容易的就被会长看上。貌似这个实验项目是上层要求做的。”

她摸了摸下巴，做出思考的状态。

“我好像忘记了什么…怎么想不起来了呢…”

突然门外传来声响，似乎有人准备开门。

“糟了，我想起来了…”

钟兰一把拉起我的手带我跑到实验室的角落，拉开柜子把我推了进去。力道之大，同时我也有些茫然，一把被摔在柜子里振的我有些懵。

她关起我面前的门，隔着门对我说。

“我忘记今天会长要来看看成果了，实在抱歉，等等她们走后我会放你出来，你先等等…”

我看着钟兰关上了柜门，四周陷入了黑暗。

“这怎么回事…我…”

头顶突然感觉有什么东西砸了下来，随后我的头上流下了浓稠的液体。应该是刚刚摔在柜子里的时候把头顶的夹层振出来了吧。

但我有些心慌，想起刚刚她对我说的话，难道这液体是刚刚的…

“钟兰，你这么早来啊…”

“会长好，我过来看看实验报告是否出错，也是以防万一…”

柜子外面传来交谈的声音，似乎有三四个人的样子，其中会长的声音有些严肃，给人一种成熟的感觉。

我感觉自己手脚有些麻痹，思维开始混乱，甚至连说话都变得困难了很多。

但我还是保持不懂，毕竟钟兰知道我在柜子里面，就算出了什么问题她也能找到并解决我的问题吧…

“对了钟兰，你的搭档在实验部等你，出现了新的情况让你过去开下会。”

“啊…会长，这，我等一下去可以吗，我有些事要做…实验药物的回收保存，完成之后我会过去的…”

“不要紧，我让她们处理，你先过去吧…这里我来处理，毕竟我也有些事要看看。”

钟兰好像没办法，只能离开实验室，留下会长和几个助理看管实验室。

听见钟兰越离越远的声响，我的心也提了起来，一切都要看造化了。而现在的我似乎也受到了影响，尽管四周很黑，但我依旧能感受到自己的身体，已经不是原来的样子了…

“帮我把实验体拿过来来，还有智慧型…”

一阵瓶子碰撞和杂乱的脚步声，随后似乎都向我这里靠近。

我有些慌忙，黑暗中摸索到一个墙壁，随后沿着它向里面靠近。

突然柜门被打开，一阵光亮晃了我的眼睛，我失去重心摔倒在地上。

“实验品是在这里吗…我记得…”

我旁边的东西被提了起来，光亮刺的我睁不开眼。

“咦，这里怎么还有一个…实验品保管的不太好啊…”

我刚想爬起来，两个柱子夹住了我的腰把我提了起来，我眯着眼睛，稍微恢复了点视力。

眼前的少女眨了眨眼。

“怎么还穿着衣服呢，实验室的奇葩怎么还有这癖好…”

她轻笑着，随后粗暴的撕开了我的衣服。

我努力挣扎，但对她来说毫无意义。

“难道是智慧型吗？可是看着也不像啊，应该有编号的吧…”

她有些戏谑的看着我，随后用指甲轻轻的撕裂了我的内裤。

“啧啧啧，实验部这些人真是厉害，这都能复制出来啊…”

看了我一眼后打开了旁边的箱子，把我丢了进去。

我十分害怕，从高空掉落后摔进了另一个漆黑一片的地方。

难道我被误认为实验品了吗，钟兰她还能认出我把我救出去吗？

我十分害怕，忘记了摔落后的疼痛，全身冰冷起来。

我该怎么办？
