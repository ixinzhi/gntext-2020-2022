# 瓶子

作者：kkkkzzzz

TID：30989

 

# 1

“我本来是一个自由的小人 有一天我迷路了 结果就被她捡了回去。而现在 我是她养的小人”

“一开始她对我很好 给了我很多自由 可是我也是有家人的人 有一次我和她说我想回家看看 她就突然暴怒了。她是那样的贪婪和不信任我 怕我跑掉 就把我关在瓶子里面”

“瓶子里面经常又闷又热 于是她把瓶口的盖子换成了纱网 总算是舒服了一点 她每天也都会给我好吃的”

“日子越来越久 一年一年地过去 逐渐我也忘掉了对家人的思念”

“我真正地成为了她养的宠物 因为我的所有食物来源 情绪来源都是她”

“我没有那么一刻地依赖一个人 我每天都等着她回来 因为只有她回来之后 我才可以从瓶子出去 才可以玩乐 才有人和我说话”

“我已经分不清了 我到底是在依赖她 还是在依赖她能给我带来的事物”

“我想起了领居家养的狗也是这样 每天都在等着主人回来 主人给它食物 主人给它快乐 没有了主人 狗就什么都不是了 没了她 我也什么都不是了”

“我大概已经扭曲了 不过没有关系”

“日子一天的一天的过去 我每天都在期待她的回来 在她离去的失落与孤独 和等她回来之后的期待和快乐之间起伏”

“我以为日子会这样一天一天下去 但是我错了”

“她是贪婪的”

“她带回来了第二个小人”

“她是那样地喜欢他 为了不让他跑掉 他的瓶子甚至都是双层的 不仅有纱网 还有塑料瓶盖 还要再在外面套一个大瓶子扭紧”

“那里面一定很闷 但不知道为什么我却感到了嫉妒 我也想被被那样对待 被瓶子关起来 被好几个瓶子套着关起来 我想被她紧紧地关起来”

“那个小人是那样地会讨她的欢心 有一天在她离开前 他管她要了好多颜料 并要求做成小小的笔 理由是在瓶子里关着很无聊 他想画画”

“她当然会满足这个小小的要求 很快就给他最好了小颜料笔 于是离开了”

“我以为他只是画画 我没想到的是他不只是画画 他要在瓶子上面画画 画了好多关于她的画 她的长发 她的脚 她的身体 她的脸 还写了一句他十分爱慕她的话语”

“她是那样贪婪 回来之后当然对这样的把戏欣喜不已”

“日子一天一天地过去 她对他越来越喜欢 对我的关注越来越少 甚至有时候她会忘了我的存在 忘了把我放回瓶子里”

“不过我会自己爬过去 拿毛线绑着重一点的橡皮泥 借重力甩进去 然后攀爬回去 我是她听话的小人”

“日子一天一天地过去 她是那样贪婪 她给他的瓶子越来越厚 瓶盖越来越紧 一个瓶子套一个瓶子”

“日子一天一天地过去 她是那样健忘 她给我的瓶子经常忘记关起来 甚至有好几天都忘了把我放回去”

“终于有一天 她对我说 让我离开”

“我可以回去看我的家人了 不用再每天生存在瓶子里面等待她了”

“她不需要我了呀原来”

“可我没有离开”

“我每天都会如往常那样呆在桌子上等她回来 我会自己弄来吃的 我很乖 我会自己进入瓶子”

“我不会离开 我也不想离开 她不需要瓶子也可以让我永远呆在她身边”

“因为她的存在 就是禁锢我的瓶子”
