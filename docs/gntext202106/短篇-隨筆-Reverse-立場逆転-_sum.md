# 【总结】短篇／隨筆 Reverse[立場逆転]

-   **春塚獲得超能力縮小**
    1.  春塚獲得縮小能力，能變小吃零食。
    2.  春塚告訴妹妹後，妹妹拿來作為把柄。
    3.  春塚對妹妹的過分行為感到不滿。
-   **春塚發現新能力Reverse**
    1.  春塚得到新能力Reverse，但無法使用。
    2.  春塚計劃利用Reverse對妹妹報仇。
    3.  春塚在妹妹回來時試驗Reverse。
-   **Reverse讓春塚和妹妹變性別**
    1.  Reverse讓春塚變成女生，妹妹變成男生。
    2.  春塚利用Reverse對妹妹進行報復。
    3.  妹妹對春塚的報復感到興奮。
