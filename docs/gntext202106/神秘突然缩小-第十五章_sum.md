# 【总结】神秘突然缩小 第十五章

    2.  表妹邀请阿天吃麻辣烫，阿天为麻辣烫妥协。

    2.  小玥是个瘦弱但高挑的小美女，房间整洁。

    2.  阿天醒来发现自己变小，且没穿衣服。

    2.  小玥在旁，阿天感到压迫时间漫长
    3.  压迫结束后，阿天感到幸运未受重伤
