“我也是，我可以为你，做到任何的事。”

是的，任何的事。

正当我们互诉倾肠的时候，邻居家终于听到了动静骂骂咧咧地从房间里冲了出来。

但我觉得他很有可能是还没睡醒，在看到我巨大的身体后第一反应不是逃跑而是破口大骂

本来一开始并不想理他，只准备等歇一会后再消除他的记忆，可他的语句越发粗俗不堪，听得我和老公都直皱眉头。

于是我坐了起来，把老公转移到我的肩膀处，冷冷地盯着那个男人。

我觉得任何人在行房事之时被打断都会有些不高兴吧，我虽然拥有了超脱人类的力量可在这种事情上还是不能免俗。

就当我准备给这个男人一点教训之时，老公凑到了我的耳边说到，

“老婆，杀了他。”

我的动作一顿，有些错愕地转过头，然而老公的眼里却平淡如水，就像是诉说了一句再普通不过的事情。

“我无法容忍对我的神不敬的行为，如果老婆大人不忍下手的话我也会去杀了他的。”

不能容忍吗，这句话让我想起了曾经，曾经我还是个那么普通的女孩的时候，那个时候老公就已经陪在了我的身边，但凡有一点对自己诋毁的话语他都会毫不犹豫地为自己出头。

曾经是他保护着我，现在既然我可以保护他了，那我又有什么理由不满足他这么一个的愿望呢。

我看向老公的眼睛越发温柔似水，可在底下的那个男人看来却充斥着恐怖，他悄悄地移动脚步就想着逃跑。

可已经晚了。

意念一动，周围所有的空气都在一瞬间忽地向他猛烈挤压，巨大的压力径直把他死死地压在了地上。

“你怎么可能跑的掉呢，不过是.....一个虫子罢了。”

我本想说一个人类，可想到旁边的老公也还是凡人之躯，所以临时改口换成了一个虫子。

毕竟除了我老公以外的基本所有人，此刻在我眼里都和虫子无异吧～

往后很久回忆起这一天，我也不知道是不是这个时候开始我的想法才开始改变。

“对的，老婆大人，就是虫子，所有的人类都应该是你脚边的虫子啊。”

老公的下体又硬了起来了，真是的，明明我只是说了一句话而已啊，那你看了我接下来的动作不是只会更加兴奋吗～

我上前一步，巨大且完美的裸体在这一刻就是最完美的行刑工具。

慢慢地伸出脚，我想要我的老公看清楚一切的细节，绝对力量下的差距，死亡前的恐惧，血液从身体四处溢出.....

啊，其实，我也非常想看呢～

“”我可以感受到在我的脚底之下，男人的尸体已经变成了一块看不出模样的糊糊，但我显然觉得这样还不够。

对着已经准备老公狡黠一笑，脚底在这一刻再度发力。

那是如同蛋壳破碎般的脆响，深不见底的裂缝自我的脚为中心向外蔓延，房子，道路，树木都在往下陷，电线断链发出咳兹咳兹的电流声，在地壳深处发出恐惧的轰鸣，如同大地哀鸣般的惨叫。

仅仅数秒后，方圆十里内自我为圆心变成了一块盆地。

我的力量有这么强吗，明明我只是按照最低力量的限度塑造了这个分身而已啊，现在看来就算是一个分身毁灭这颗星球也绝对绰绰有余吧。

其实仔细想想现在的人类在我本体的面前到底是个什么样的存在呢。别说虫子了，就连分子也不如啊，对自己而言。

我就是人类的世界吗

啊，脸上又烧起来了，之前刻意地不去思考，现在光是想一想就兴奋起来了吗。

当然我可不想在老公面前表现出太没见过世面的样子，所以我还要做出一份不满的模样。

“一个脚趾毁灭城市果然太勉强了吗～”

“用一个脚趾毁灭城市吗，老婆大人你的力量可以做到这种程度吗，那我还想看老婆大人.....啊，不 我不可以.....”

“可以哦。”

我微笑地把老公捧到手心，狠狠地亲了他一下

“你可以命令我做任何事，只要你能想象出来的我都可以为你做到，我说过了吧，我可以为你做到任何事。”

情绪从未有一刻像此时这么高涨，那种可以为心爱之人做到任何事带来的成就感难以想象，此前竟然还会觉得成为神无所谓，真的是，

明明是这个世界上最让人兴奋的事啊～

接着我就这么静静地看着老公。

不行，思考的样子也好帅，再亲一口。

嘻嘻～

但这次他没有像上次那样射出来，不过看样子也是忍得非常辛苦啊。

终于过了好一会，他抬起头对着我说道，

“老婆大人，我觉得你现在的身高太小了。”

“哦？太小了吗。”

我微笑着再度巨大化自己的身体，还刻意张开双手让老公感受升降机一般的高低落差感，夜空中的寒风吹得他瑟瑟发抖，直到我升起我的手指才好了一点。

四百米，这是我此刻的身高，本就在刚才被我震碎的建筑此刻更是如同碎片般散落。

而我的老公正紧张地扶住我的手指，小心翼翼地朝下看去，然后忍不住发出惊叹。

“就算在这么高，刚才老婆造成的裂缝也在那么远的地方啊。”

“那是当然～我的能力绝对超乎你的想象。”

不过我们所居住的地方位于刚开发没多久的景点别墅，刚才的邻居九十我们附近唯一的住户了。

不然我也不敢玩起这种游戏，毕竟真的一脚震碎一座城这种事情多多少少还是有点于心不忍。

看着老公兴奋地如同孩子般的神情我也有些跃跃欲试起来，

“我的力量可远远不止如此哦～”

就像是迫不及待展示玩具的孩子一般，我这么对着老公说道，

“那老婆大人你是打算.......”

“我不只是能破坏一切，我还可以修复一切哦～”

脚趾再度轻点，眼前的一切就仿佛点起了回放键一般，大地愈合，草木生长，就连刚刚被我的脚掌碾碎的房子都回复如初，就在我的脚趾旁的领居家内，那个有些秃头的男人睡得跟头死猪一样，浑然不知道他刚刚死了一次。

“你看，不管动作有多大我都可以恢复成原本的模样，甚至就算被人看见了我都可以修改记忆，”

我兴奋地自说自话，全然没有注意到老公慢慢冷下去的眼神。

“所以不管你想做什么，我都可以.......”

“够了！”

我轰鸣般的声音乍然而止，我僵硬地低头，一时没有反应过来，我们两人陷入了死一般的沉寂。

大概过了好几分钟。

手心的老公深吸一口气，讪笑着挠了挠头，抬起头对我说道，

“不好意思啊，老婆大人，我今天.....可能有点累了，要不今天就先到这里，好吗？”

“啊....好的，嗯，今天的确是很累了呢，是的，今天太累了，要好好休息呢，没错，先回去好好休息，对，休息一下 休息一下就好了。”

我忙不送地点头，赶忙缩回了原本的体型。

夜空寂寥如烟，路灯昏沉沉地亮着，方才我们所做的一切都宛若幻觉。

我有些恍惚，呆呆地看着老公从我怀里下来，再一个人走回屋子，没再和我说一句话。

一阵冷风吹过，我这才意识到从刚才开始我就一直身无寸缕。

刚才自己到底怎么了，变得完全不像自己。

心里面忽的无聊了下来，我走回房屋，就这样把大门关上。
